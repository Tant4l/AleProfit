# AleProfit

Webapp do obliczania *rentowności zamówień* w serwisie e-commerce Allegro.

## Funkcje aplikacji

- Przeprowadza łączenie konta Allegro z aplikacją oraz autentykację za pomocą tokenów.
- Pobiera i uporządkowuje dane dotyczące zamówień oraz płatności w Azure SQL Database.
- Wykonuje obliczenia na podstawie danych i indywidualnych ustawień w celu obliczenia "czystego profitu" transakcji.
- Przedstawia dane w przystępny sposób za pomocą dashboardu w Azure Static Web Apps.

## "Tech stack"

- Azure SQL Database
- Azure Functions
- T-SQL
- Allegro API
- Azure Web Apps
